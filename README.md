# JAX LLM examples

A collection (in progress) of example high-performance large language model
implementations, written with JAX.

Current contents include:

* [DeepSeek R1](deepseek_r1_jax/)
* [Llama 4](llama4/)
* [Llama 3](llama3/)
* [Qwen 3](qwen3/)
